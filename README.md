# Loan Approval Prediction

This is my first end-to-end Machine Learning project. The application predicts whether a loan application is likely to be approved based on the details entered by the user.

I built the machine learning model using Scikit-learn and created the web application using Streamlit. The project is deployed online using Streamlit Community Cloud.

## Features

- Predicts whether a loan will be approved or rejected
- Simple and easy-to-use interface
- Built using a trained Random Forest model
- Deployed online using Streamlit

## Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. Handling Missing Values
4. Feature Encoding
5. Train-Test Split
6. Model Training
7. Model Evaluation
8. Hyperparameter Tuning
9. Model Serialization using Pickle
10. Deployment with Streamlit

## Models Tried

I trained and compared multiple machine learning models:

- Logistic Regression
- Decision Tree
- Random Forest

After comparing their performance, I selected the Random Forest model for deployment.
